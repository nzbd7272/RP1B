# RP1B
Key Features:
Singleton Extraction: Isolates the "rawest" forms of mutation by targeting alleles present in only one isolate.

Density-Based Recombination Filtering: Uses a 2,000 bp sliding window to exclude clustered polymorphisms.

Trinucleotide Contextualization: Maps every SNP to its 5' and 3' flanking bases.

Genomic Background Normalization: Adjusts mutation rates based on the total abundance of specific bases/triplets in the reference genome.

Visual Analysis: Includes R scripts for generating publication-ready, color-coded bar charts
